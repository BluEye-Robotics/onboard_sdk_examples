Blueye SDK

The Blueye SDK provides a set of libraries and tools for controlling and interacting with Blueye drones. It enables developers to communicate with the drone, access sensor data, and control various aspects of its behavior.

Getting Started

To start using the Blueye SDK, follow these steps:

  1. Install the Blueye SDK by downloading the latest release.

  2. Include the necessary headers and libraries in your project. For example, you may need to include the following:

    #include <blueye/drone/Drone.hpp>
    #include <blueye/imu/ImuData.hpp>

  3. Create an instance of the blunux::drone::Drone class to interact with the drone:

    blunux::drone::Drone drone;
    blunux::drone::Drone
    Class representing a drone.
    Definition: Drone.hpp:28

  4. Use the available methods to control and monitor the drone. For example, you can set the light intensity, retrieve sensor data, or control the thrusters:

    drone.set_light(0.5); // Set light intensity to 50%
    auto imuData = drone.get_imu(); // Retrieve IMU data
    drone.set_thrusters(thrustersSetpoint); // Control thrusters
